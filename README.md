# ALU Machine Learning

This repository contains projects and assignments for the ALU Machine Learning curriculum.

## Mathematics for Machine Learning - Plotting

This project demonstrates various plotting techniques using matplotlib in Python.

### Requirements

- Python 3.5+
- numpy 1.15+
- matplotlib 3.0+

### Installation

```bash
pip install --user matplotlib==3.0
pip install --user Pillow
sudo apt-get install python3-tk
```

### Project Structure

```
math/plotting/
├── 0-line.py          # Line graph (y = x^3)
├── 1-scatter.py       # Scatter plot (Height vs Weight)
├── 2-change_scale.py  # Logarithmic scale (C-14 decay)
├── 3-two.py          # Multiple lines (Radioactive decay)
├── 4-frequency.py    # Histogram (Student grades)
├── 5-all_in_one.py   # Combined plots
└── 6-bars.py         # Stacked bar chart
```

### Usage

Each script can be run independently:

```bash
python3 0-line.py
python3 1-scatter.py
python3 2-change_scale.py
```
